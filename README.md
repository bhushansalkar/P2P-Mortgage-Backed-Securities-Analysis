# P2P-Mortgage-Backed-Securities-Analysis


## Project Overview

This project focuses on analyzing mortgage loan data to understand **loan performance, borrower behavior, and financial risk indicators**. The objective is to transform raw loan data into meaningful insights that help monitor portfolio performance and support better lending decisions.

Using **data preprocessing, feature engineering, and exploratory data analysis**, the project identifies patterns in loan performance indicators such as prepayment risk, borrower credit characteristics, and interest rate structures.

An interactive **Power BI dashboard** was developed to visualize key metrics and provide insights for financial analysis and risk management.

---

## Objectives

- Perform **data preprocessing and feature engineering** on mortgage loan datasets
- Analyze **loan performance indicators**
- Identify **financial risk factors affecting loan portfolios**
- Build **interactive dashboards** for monitoring loan performance
- Generate **data-driven insights for risk management and lending decisions**

## Key Analysis Areas

The project analyzes several important loan portfolio indicators:

- Loan Prepayment Risk
- Borrower Credit Characteristics
- Interest Rate Structures
- Loan Performance Metrics
- Portfolio Risk Indicators

These indicators help evaluate **investment risks and portfolio performance**.

---

## Dashboard Features

The Power BI dashboard includes the following insights:

### Loan Performance Overview
- Total loan portfolio metrics
- Active vs closed loans
- Performance indicators

### Borrower Credit Analysis
- Credit score distribution
- Borrower risk segments
- Default risk indicators

### Prepayment Risk Analysis
- Loan prepayment patterns
- Interest rate impact on loan behavior

### Portfolio Risk Monitoring
- Identification of high-risk loans
- Financial indicators impacting investor returns

---

## Key Insights

- Borrower credit characteristics significantly influence loan performance.
- Interest rate structures impact prepayment behavior.
- Certain loan segments present higher financial risk for investors.
- Data visualization helps quickly identify portfolio trends and risk patterns.
